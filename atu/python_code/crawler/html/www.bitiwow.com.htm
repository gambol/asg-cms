看了MT之后,心底的热情被激发了,装了游戏.卸载所有播放器,和游戏.因为实在是过太久了,我都已经忘了怎么选服务器,就是随便跟着系统提示,进了一个服务器.建了一个法师号,兴致勃勃的练着级.
某一天,在逛论坛,突然兴致起,在论坛上发了一帖“我是一个法师”,没想后面跟帖的还挺多,原来我们这个小城镇,这么多人玩魔兽私服,大家建了一个群,加对WOW私服有兴趣的朋友.然后一帮人整天在群里商量着,去哪个区,哪个服务器,就在我要确定去五区的时候,他进了群,然后莫名其妙的,我把本来说好的那几个人抛弃了,一个人跟着他去了十区,那时候我们不认识,我也不知道他的职业他的种族.他就是告诉了我十区哪个服务器.当天晚上我就在这个服务器建了号,冥冥之中的安排,本来我一直很中意亡灵的,而且很喜欢SS,想好要建个亡灵SS,却不知道出于什么原因,我选了血精灵法师.在我5、6级的时候有个牛头战士过来找我,给了我包包和G,他说是他的朋友,后来我才知道这个号也是他的.一次我在银月迷路了,我才见到他,一个血精灵QS,好帅,那是我对他的第一印象.后来我就这么练着级,那时候他们开荒TOC,ICC并没有太多时间照顾我,我就凭着内测的记忆摸索着升级.
我们的感情在法师和骑士身上展开,他在游戏里很照顾我,他叫我妹子,把我加进公会,把我介绍给公会成员,带我做任务,下副本升级.他说他从来没有这么带过一个人.后来我才知道,他是我们会长,我们公会也算是服务器几大公会之一.我到80级之后,他会带我打ICC,全程都照顾着我,有出法系的装备他会在团队说：这个你们懂的.然后我的包包就会多一件我觉得很好的装备.在我搞不清状况,快要OVRE的时候,他会给我上保护之手,虽然我是一个法师,但是我经常记不起用冰箱可以保护自己.他是一个防骑,有时候在治疗来不及加血的时候,他会给我加上一把.他会在没公会活动的日子,陪着我去跑地图,会陪着我去刷10级、20级、30级时的副本.因为我说我要做成就,他就陪着我到处拿“元祖硬币”.公会里的人说他,开始沦陷了.那个时候,我们连对方长什么样都不知道.除了在游戏里,我们在QQ里也想谈甚欢,因为两个人的工作时间关系,我们白天基本都窝在QQ里,聊情感生活,聊魔兽世界私服,聊论坛帖子,天南地北,什么都聊.我们的性格,对感情的执着都非常的相似.
自然的,我们相互换了手机号码,也相互看了照片,虽然我们在同一个城市,但是我却不希望两个人能见面,我觉得在游戏里,感觉挺好的,其实,在我看他的照片的时候,我完全没好感,我喜欢的是QS,而不是他,当时,我觉得我是不会接受他的. 第一次见面,是一个下雨天,他在我上班的必经之路等着我,我不知情,站在公交车站等着车,结果他在我面前停下了车,很无奈的,我上车了,当时我真感觉手足无措,我不知道怎么处理这种情况,上车之后两个人无语,我都不敢看他,看的出他也紧张吧,中途还熄火了.
后来他告诉我,他已经等我好几次,只是都没等到.那时候心里有感动也感觉别扭.QQ还是聊着,但是我更多的喜欢在游戏跟他聊天,确切的说我是喜欢跟QS聊天,我喜欢跟在QS后面,看着他.
游戏里,生活上他一如既往的对我好着,游戏里,只要有我在,他就会寸步不离,直到公会活动开始.生活上一有时间他就送我上班,接我下班,他不会提前通知我,只在路口等着.他说喜欢给我惊喜,其实,每次我都能看到他车停那边,只是我装着不知道.他会记得我说的每句话,我喜欢吃松露巧克力,他就隔断时间给我买一盒,直到我说我真不吃了,很胖了,但是他还买,只是把间隔延长了.他知道我喜欢吃必胜客,他在YY里建了一个频道“必胜客”成为我和他的私聊区.每个星期会带我出去吃必胜客,因为我喜欢,他说他也喜欢了,他说以前他从来没想过自己会吃这个.
直到今天2011年3月10日,整整202天,我在他爱上我之后深深的爱上了他,明知道我们是不可能的,但是仍爱的无法自拔.我曾经痛苦的想要离开游戏,离开他.他说如果我离开了游戏,他立马删号,我问他能舍弃一手创建出来的公会吗,他说如果魔兽私服没有了你,就不再有任何意义.
所以我们约定,既然已经注定不能相爱与现实,就让我们在WOW私服里相爱一辈子,永远守着能源舰,守着部落,守着公会,哪怕最后只剩我们两个人.
